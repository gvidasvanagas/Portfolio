Bootstrap Components Used

  • Navbar (responsive collapse toggler, sticky-top)
  • ScrollSpy (highlights the active nav section while scrolling)
  • Grid system (containers, rows, columns for layout and responsiveness)
  • Buttons (primary/outline variants, button groups)
  • Cards (used for contact panels/details)
  • Accordion (Collapse) for Work Experience entries
  • Tables (education modules/grades)
  • Modal (Contact Modal with contact info)
  • Toast (feedback on contact form submission – demo)
  • Forms + validation styling (required fields and invalid feedback)
  • Utility classes (spacing, typography, flex, shadows)

Dynamic Features Added (JavaScript)

  • Education year filter: Year 1–4 buttons switch displayed modules/grades. Year 3 automatically hides Semester 2 to reflect placement.
  • Smooth scrolling with correct navbar offset: clicking nav links scrolls smoothly to the section while accounting for the sticky navbar height.
  • Active section highlighting: ScrollSpy is initialised/re-initialised with a dynamic offset so the navbar highlights the correct section as you scroll.
  • Back-to-top button: appears after scrolling down and returns the user to the top smoothly.
  • Contact form validation + toast: validates required fields locally and shows a Bootstrap toast message (demo).
  • Automatic footer year: updates the copyright year dynamically.
